Position

The position property - a property that allows us to position elements in ways that may not be in the document's usual code flow.

This is our stickied header.

DO NOT ABUSE POSITION!!

  • Static - Position: Static is the default value of position for every element.
  • Relative - The element is positioned relative to its normal position in the document's code flow.
  • Absolute - The element is positioned relative to its closest positioned ancestor.
  • Fixed - The element is positioned relative to the viewport.
  • Sticky - The element flips between relative and fixed until the user's coordinates are met.
